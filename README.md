# Astrosage Call Center Optimization Dashboard

## Objective
Analyze monthly sales performance using Excel.
My excel project title is "Astrosage Call Center Optimization" and astrosage is astrology consultation platform. The objective is to analyzed historical call data, performance metrics, and customer satisfaction in order to design proactive strategies for improving operations. AstroSage has received an investment and aims to optimize its call center operations. 

## Features
- Pivot Tables
- Charts & KPIs
- Data Cleaning

## Tools Used
- Microsoft Excel

## Insights
- Calls, though only ~30% of volume, generated over 80% of revenue and had higher ratings (3.5/5 vs 1.3/5 for chat)
- Chat completion was very low (28%), dragging satisfaction
- Best call success windows were 5–9 AM and 2–5 PM (completion ~49%), while late nights had high failures (down to 8–15%)
- The App channel contributed 66% of revenue despite fewer interactions—showing stronger monetization
